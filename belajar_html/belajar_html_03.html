Belajar Menggunakan Elemen tag HTML < HR > (Horizontal Rule)

HTML elemen tag < hr > (horizontal rule) merupakan elemen pada HTML yang digunakan untuk membuat garis datar memanjang secara horisontal.

Elemen < hr > sering digunakan saat terjadi perubahan tema bahasan pada level elemen paragraf sehingga diperlukan sebuah garis horisontal sebagai pemisah dengan bagian di bawahnya.
